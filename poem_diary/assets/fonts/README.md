# Font Assets Placeholder

Bu klasör, uygulamada kullanılacak özel fontları içerir.

## Gerekli Fontlar

### 1. PlayfairDisplay (Serif - Şiirler için)
Fontları şu kaynaktan indir: https://fonts.google.com/specimen/Playfair+Display

Gerekli dosyalar:
- `PlayfairDisplay-Regular.ttf`
- `PlayfairDisplay-Italic.ttf`
- `PlayfairDisplay-Bold.ttf`

### 2. Montserrat (Sans-Serif - UI Yazıları için)
Fontları şu kaynaktan indir: https://fonts.google.com/specimen/Montserrat

Gerekli dosyalar:
- `Montserrat-Regular.ttf`
- `Montserrat-Italic.ttf`
- `Montserrat-Medium.ttf`
- `Montserrat-Bold.ttf`

## Kurulum Adımları

1. Google Fonts'tan fontları indirin
2. TTF dosyalarını bu klasöre (`assets/fonts/`) kopyalayın
3. Uygulamayı çalıştırın - fontlar otomatik yüklenecektir

**Not**: Font dosyaları olmasa bile, uygulama çalışacaktır çünkü Google Fonts paketinde yedek fontlar vardır.
